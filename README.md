# SYNOPSIS

Installs git client and makes some basic configuration.

# INSTALL

    $ sparrow plg install git-base

# USAGE

    $ sparrow plg run git-base --param email=melezhik@gmail.com --param name="'Alexey Melezhik'"

This command will installs git client and set git configuration for email and name parameters. 

# Platfroms supported

* CentOS
* Ubuntu
* Debian
